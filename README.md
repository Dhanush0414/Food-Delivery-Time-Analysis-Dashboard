# Food Delivery Time Analysis Dashboard

## Overview

This project is an interactive dashboard built using Python and Streamlit for analyzing food delivery data. It helps users explore datasets, visualize delivery patterns, and generate insights through Exploratory Data Analysis (EDA).

## Features

- Dataset Preview
- Missing Value Analysis
- Statistical Summary
- Delivery Time Histogram
- Correlation Matrix
- Correlation Heatmap
- Dataset Insights

## Tech Stack

- Python
- Pandas
- NumPy
- Streamlit
- Matplotlib
- Seaborn
